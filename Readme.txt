To run with default 'settings' just run as 'java Test' on a terminal, the default test file name is 'TestData.blf' and the default confidence threshold value is 0.75. Confidence threshold is in range of [0, 1] where 0 - is no match, 1 - is full target match. 

You can also specify different input file and confidence threshold values as follows - 'java Test <input_file_name> <confidence threshold> '.
